# 挂钩记忆系统

## Peg System of Memory

### 什么是挂钩记忆法？

挂钩记忆法（Peg System）是一种为**数字**分配固定图像的记忆系统。

**与链式记忆法的区别**：
- 链式法：项目之间互相连接
- 挂钩法：每个项目与一个固定的"挂钩"连接

---

### 数字形状挂钩法

这是最简单的挂钩系统，根据数字的形状来创建图像：

| 数字 | 形状挂钩 | 图像 |
|------|----------|------|
| 1 | 铅笔 | 一根细长的铅笔 |
| 2 | 天鹅 | 天鹅的脖子 |
| 3 | 耳朵 | 人的耳朵侧面 |
| 4 | 帆船 | 帆船的帆 |
| 5 | 钩子 | 鱼钩或挂钩 |
| 6 | 高尔夫球杆 | 球杆和球 |
| 7 | 悬崖 | 陡峭的悬崖 |
| 8 | 雪人 | 两个球堆成的雪人 |
| 9 | 气球 | 气球和线 |
| 0 | 鸡蛋 | 椭圆形的鸡蛋 |

---

### 使用示例

**记忆列表**：1.苹果 2.香蕉 3.橙子 4.葡萄 5.西瓜

1. **铅笔 + 苹果** → 想象铅笔戳穿了一个大红苹果
2. **天鹅 + 香蕉** → 天鹅叼着一根香蕉在飞
3. **耳朵 + 橙子** → 耳朵上挂着一个橙子当耳环
4. **帆船 + 葡萄** → 帆船的帆是由葡萄串组成的
5. **钩子 + 西瓜** → 钩子勾住了一个大西瓜

---

### 练习

用数字形状挂钩法记忆以下内容：

1. 猫
2. 书
3. 眼镜
4. 手机
5. 水杯

**提示**：先熟悉 1-5 的挂钩图像，然后与项目建立联系。

### 进阶：主要系统（Major System）

主要系统将数字转换为辅音，然后组成单词：

| 数字 | 辅音 | 记忆提示 |
|------|------|----------|
| 0 | s, z, soft c | zero 首字母 |
| 1 | t, d | 一笔写成 |
| 2 | n | 两笔写成 |
| 3 | m | 三笔写成 |
| 4 | r | fouR 尾字母 |
| 5 | l | 罗马数字 L=50 |
| 6 | j, sh, ch, soft g | 形状像 6 |
| 7 | k, hard c, hard g | 两个 7 组成 K |
| 8 | f, v | 手写体 f 像 8 |
| 9 | p, b | 9 旋转是 p，镜像是 b |

---

> "挂钩系统的优势在于：挂钩是永久的，可以反复使用。"

掌握了挂钩系统，你就可以轻松记忆任何有序的信息！

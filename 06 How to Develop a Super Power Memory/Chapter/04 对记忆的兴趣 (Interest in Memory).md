强调了“兴趣”是记忆训练的第一步，并详细阐述了兴趣、观察与记忆之间的因果关系。

以下是本章的核心内容讲解：

1. 著名的“公交车司机”测试

作者通过一个具体的测试证明，我们只会记住自己感兴趣的信息：

- **测试过程**：读一段关于公交车上下客过程的文字，包含多次停站和乘客人数的变化。
- **结果分析**：大多数人能算出最后剩下多少乘客，但无法回答“公交车一共停了几次”或者“司机的名字叫什么”。
- **结论**：因为读者预设“人数”是重点，所以只对人数产生了**兴趣**并给予了**注意**；而由于对停站次数或司机姓名缺乏兴趣，大脑根本没有对这些信息进行**观察**和记录。注：司机的名字其实就是读者自己的名字，因为文章第一句写的是“**你**在开车”。

2. 记忆的核心法则：强制兴趣

作者指出，记忆与观察是并行的，而兴趣是它们的动力：

- **不可逾越的规律**：你不可能记住任何你没有观察到的事物；而你很难观察或记住任何你没有兴趣的事物。
- **强制策略**：要立即改善记忆，必须**强迫自己产生兴趣**去观察你想记住的事物。
- **动力论**：没有动机，就几乎没有忆念。

3. 信心：堵住记忆的“筛子”

除了兴趣，**信心**对记忆也至关重要：

- **筛子类比**：如果你总是说“我记性不好”，就像是在记忆的“筛子”上多钻一个洞；如果你充满信心地说“我会记住它”，就是在堵住这些洞。
- **信任记忆**：记忆力喜欢被信任，你越信任它，它就越可靠。

4. 拒绝“笔头依赖”

作者极力反对用纸笔代替大脑：

- **负面影响**：写下来往往是忘记的开始，因为写下来的动作意味着你拒绝或懒于在当下进行记忆。
- **不进则退**：由于缺乏练习和信任，过度依赖笔记会使记忆力因退化而变得更糟。

5. 证明：你其实拥有“好记性”

作者通过生活中的例子证明，每个人都有潜力，只是缺乏兴趣：

- **学生案例**：有的孩子记不住历史或代数，却能如数家珍地背出所有棒球运动员的击球率，这证明他们的记忆力没问题，只是对学校功课缺乏兴趣。
- **社交案例**：有些女性抱怨记不住事，却能详细描述几周前见过的朋友穿了什么衣服、发型细节甚至是大概年龄，这再次证明兴趣所在之处，记忆力就极其敏锐。
